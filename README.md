# To-Do List

A simple and interactive To-Do List web application built with **HTML**, **CSS**, and **JavaScript**, featuring task management using checkboxes, expandable/collapsible accordion sections, and localStorage for saving tasks between sessions.

---

## How It Works

- Users type a task in the input field.
- The **Add** button becomes enabled when input is detected.
- Once added, tasks appear with a checkbox inside the "Tasks" accordion.
- The app remembers tasks even after the browser is closed or refreshed.
